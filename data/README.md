# Data

All datasets for this paper are publicly archived on **Harvard Dataverse**.

**DOI:** [10.7910/DVN/V5NSZX](https://doi.org/10.7910/DVN/V5NSZX)

---

## Contents

| File | Description |
|---|---|
| `foreclosure_thresholds.csv` | Computed x* values across sigma_R and gamma calibrations |
| `monte_carlo_paths.csv` | Simulated positional gap trajectories (N=5,000 paths per sigma_R) |
| `theorem2_calibration.csv` | Volatility-delay replication table (sigma_R 0.05 → 0.20) |

## Replication

All datasets are generated by the code in `/code/`. To reproduce:

```bash
pip install -r requirements.txt
python code/bvp_solver.py
python code/monte_carlo_simulation.py
```

See [REPLICATION.md](../replication/REPLICATION.md) for full instructions.
